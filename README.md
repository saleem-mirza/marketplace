# zethian marketplace

A Claude Code plugin marketplace. **Generated** — this repo carries only prebuilt,
source-free plugin bundles (catalog + hooks + rules + binaries), no source code.

## Install

```sh
/plugin marketplace add saleem-mirza/marketplace
/plugin install permcheck@zethian
```

### Homebrew (permcheck CLI)

To install the `permcheck` binary standalone (macOS, Apple Silicon + Intel):

```sh
brew install saleem-mirza/tap/permcheck
```

Served from the dedicated tap
[saleem-mirza/homebrew-tap](https://github.com/saleem-mirza/homebrew-tap).
